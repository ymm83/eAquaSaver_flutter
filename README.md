## Building and debugging
[Android device side] Go to Adroid conf -> System -> Dev options -> Wireless debug -> Pair device with pairing code

[Dev terminal]
  adb pair DEVICE_IP:DEVICE_PORT_FOR_PAIRING PAIRING_CODE
  adb connect DEVICE_IP:DEVICE_PORT

DEVICE_PORT_FOR_PAIRING is under "Pair deive with pairing code" along with PAIRING_CODE

DEVICE_PORT is in IP address & Port
